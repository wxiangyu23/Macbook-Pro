# 这是一个学生和老师信息管理系统

项目简介
一个基于 Linux 平台的终端命令行学生管理系统，支持学生信息管理、教师信息管理、权限控制和数据持久化存储。

开发环境
操作系统: Linux

编译器: GCC

调试器: GDB

构建工具: Make

功能模块
1. 学生信息管理
基本信息
学号 (int, 唯一)

姓名 (char[128])

密码 (char[32])

年龄 (int)

班级 (int)

成绩信息
数学成绩 (int)

C语言成绩 (int)

语文成绩 (int)

名次 (int, 自动计算)

功能列表
✅ 添加学生: 录入基本信息，自动计算名次，学号唯一性校验

✅ 删除学生: 按学号删除学生信息

✅ 修改学生: 更新学生信息

✅ 查找学生:

按学号查找

按姓名查找

按班级群体查找

✅ 显示学生列表:

按学号顺序显示

按名次显示

按学科成绩顺序显示（显示该科平均分）

✅ 数据持久化: 信息保存至文件，启动时自动加载

2. 教师信息管理
教师信息
教员序号 (int, 唯一)

教员姓名 (char[128])

教员密码 (char[32])

所任班级 (int)

功能列表
✅ 教师信息添加、删除、修改

✅ 教师信息显示、保存

3. 登录系统
用户类型
学生用户: 学生账号登录

教师用户: 教师账号登录

超级管理员: 固定用户名 admin

安全特性
✅ 用户名密码验证

✅ 登录失败3次自动退出

✅ 个性化欢迎界面

4. 权限管理
超级管理员权限
🔧 完全的学生信息管理

🔧 完全的教师信息管理

🔧 系统所有功能访问

教师权限
👨‍🏫 查看个人信息

👨‍🏫 修改个人密码

👨‍🏫 查看所有学生信息

👨‍🏫 修改学生成绩信息

学生权限
👨‍🎓 查看个人信息

👨‍🎓 修改基本信息

项目结构
text
student_management_system/
├── main.c              # 主程序入口
├── menu.c              # 菜单界面处理
├── student.c           # 学生信息管理实现
├── student.h           # 学生相关头文件
├── teacher.c           # 教师信息管理实现
├── teacher.h           # 教师相关头文件
├── tooler.c            # 工具函数实现
├── tooler.h            # 工具函数头文件
├── common.h            # 公共定义和常量
├── Makefile            # 项目构建配置
└── README.md           # 项目说明文档
数据文件
students.dat - 学生信息数据文件

teachers.dat - 教师信息数据文件

编译运行
编译项目
bash
make
清理编译文件
bash
make clean
运行程序
bash
./student_management
调试模式
bash
make debug
gdb ./student_management
使用说明
启动系统
运行程序后显示登录界面

选择用户类型（学生/教师/管理员）

输入用户名和密码

登录成功后进入对应功能菜单

功能操作
使用数字选择菜单选项

按照提示输入相关信息

修改操作会提示保存确认

系统退出时自动保存数据
